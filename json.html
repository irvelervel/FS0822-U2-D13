Appunti sul formato JSON

JSON: JavaScript Object Notation

JSON è il formato di interscambio dati nelle chiamate moderne HTTP

Lo useremo ogni volta che avremo la necessità di recuperare dati dall'internet o se avremo il bisogno di inviare dei dati a nostra volta al fine di venire memorizzati in un database passando per un server

JSON non è un formato "inventato"; è semplicemente la notazione letterale degli oggetti in JS

Qualche regola speciale: SEMPRE doppi apici """" e NON apici singoli, le coppie chiave/valore saranno separate da virgola a parte l'ultima, i valori di ogni proprietà possono essere stringhe, numeri, booleani, array, oggetti, etc.
